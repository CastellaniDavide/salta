# Description
This is the salta for cpp files.

## Required
 - a cpp compiler (eg. gcc for Linux)
 

## Directories structure
 - .github
   - ISSUE_TEMPLATE
     - bug_report.md
     - feature-request.md
 - bin
	 - salta.cpp
 - doc
   - LICENSE
   - README.md
 - log
	 - trace.log
   
### Execution examples
 - gcc salta.cpp -o salta && ./salta
